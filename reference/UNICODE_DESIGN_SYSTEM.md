# ROGT SHOPZZZ Unicode Design System

## Purpose

This document is the OFFICIAL Unicode Symbol Design System for ROGT SHOPZZZ.

Whenever you design, redesign, or improve any Discord Embed, Dashboard, Product Browser, Product Preview, Ticket, Payment, Statistics, Category Manager, Product Manager, or any future UI, you MUST follow this document.

Never invent your own emoji style.

Never randomly choose symbols.

Always use symbols from this document.

Visual consistency is more important than creativity.

------------------------------------------------------------

The UI must always feel

• Premium

• Modern

• Minimal

• Fantasy

• Luxury

• Clean

• Professional

Avoid colorful Discord emojis whenever a Unicode symbol can achieve the same purpose.
